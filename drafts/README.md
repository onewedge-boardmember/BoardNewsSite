# drafts（非公開・ローカルのみ）

このフォルダは **Git にコミットしません**（`.gitignore` 済み）。

## 使い方

1. 議事録や活動メモをここに置く（例: `2026-05-21-meeting.md`）
2. AI に渡して `posts/` 用の HTML を生成する
3. 公開前チェックリスト（リポジトリ直下 README 参照）を確認
4. 問題なければ `posts/` に HTML を追加し、`index.html` の一覧を更新して push

## 原稿の書き方（AI 向けテンプレ）

```markdown
# タイトル
- 日付: YYYY-MM-DD
- カテゴリ: 会議 / 訪問 / 調査 など

## 概要（1〜2文）

## ハイライト（箇条書き3〜5）

## 詳細（公開してよい範囲のみ）

## 次のアクション（任意）
```
