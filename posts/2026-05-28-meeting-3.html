会議 2026年5月28日

第3回定例MTG — 柱別の連携と次の一手

前回の振り返りに続き、柱 A・B・C それぞれの方針と進め方を共有。営業・教育・新規事業の連携と、定例での共有のしかたを整理しました。

会議の概要

ハイライト
  • 柱 A — 週次定例での営業数字・プロパー／パートナー営業の共有ルールを確認
  • 柱 B — 教育方針の4本柱と、エンジニア同士の相談の場の設計を共有
  • 柱 C — 年間計画・アイディア箱・ビジネスチャンス募集など新規事業の進め方を協議
  • 柱間の依頼事項（A→B、B→C）をすり合わせ、次の一手を明確化

柱 A — 営業と週次共有

柱 A の定例ミーティングでは、現状の数字を共有したうえで、課題と次の一手を全員で揃えることを目的に進めます。

  • 営業数字の共有 — 実績と目標の達成状況を共有し、テコ入れが必要な箇所を同じ認識に
  • プロパー営業 — 案件の進み具合・受注見込み・ボトルネックと対応方針を報告
  • パートナー営業 — 各パートナーの動きと案件状況を共有

販管費の進捗についても共有しました（具体額は社内限定）。マッチング作業の体制強化やツール活用など、営業を支える協力のお願いも共有されています。

柱 A から柱 B への依頼

  • 全プログラマーのスキルアップ — 営業数字を伸ばすための技術力向上の継続・強化
  • キャリア・案件に関する悩みや相談へのフォローアップ
  • 面談力の強化 — 案件参画の前提として、面談の質を高める取り組み

柱 B — 教育とエンジニア支援

教育方針を4本柱で整理しました。学習の向上、面談力の強化、不安・悩みのフォローアップ、採用です。

  • エンジニア同士で技術的な不安や案件選択を相談できる場の設計を検討
  • 社員満足度アンケートの実施
  • 進捗・成果は定性的な指標も含め、意識して見える化していく方針

柱 B から柱 C へは、新規サービスに微経験のエンジニアも参画しやすい形にしてほしいという依頼がありました。

柱 C — 新規事業とアイディア募集

柱 C の年間計画と、新規事業の進め方について共有しました。

  • アイディア箱の設置 — メンバーからの提案を受け付ける仕組みを整備
  • ビジネスチャンス・エピソードトーク — 過去の経験からビジネスアイディアを募集
  • 柱 C のオープン MTGを継続開催
  • 学生（インターン・バイト）による開発協力の可能性を検討（柱 B との親和性も含めて議論）

このあと

各柱の定例と依頼事項に沿って、週次共有・教育・新規事業の取り組みを進めます。 決まったことは、引き続き本サイトでお伝えします。
